ÁLBUM COPA 2026 — PWA INSTALÁVEL

ARQUIVOS
- index.html: aplicativo principal
- manifest.webmanifest: dados de instalação
- sw.js: funcionamento offline
- icons/: ícones do aplicativo

COMO PUBLICAR
1. Envie a pasta inteira para um serviço HTTPS, como Vercel, GitHub Pages ou Netlify.
2. Abra o endereço publicado no Chrome do celular.
3. Toque em “Instalar” ou no menu ⋮ > “Adicionar à tela inicial”.

IMPORTANTE
- A câmera do navegador só funciona em HTTPS ou quando o app está instalado.
- O reconhecimento automático usa Tesseract.js e precisa de internet na primeira utilização.
- Caso a leitura da câmera não identifique o código, use “Digitar código”.
- O progresso é salvo no próprio celular pelo localStorage.
- Toque em qualquer código para alternar entre “faltando” e “já tenho”.
